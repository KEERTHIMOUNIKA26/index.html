HTML IMPORTANT QUESTIONS

1.What is HTML?

HTML (HyperText Markup Language) is the standard language used to create and structure web pages. It defines the content and layout using tags. It helps display text, images, links, tables, and more on browsers.

2. Markup Language vs Programming Language – What is the difference?

A markup language (like HTML) structures and presents data but doesn't perform logic or calculations. A programming language (like Python or JavaScript) can perform logic, loops, and computations. In short, markup is for layout; programming is for functionality.

3. What are the basic tags in HTML?

• html: Root of an HTML document. • head: Contains meta information, title, styles. • body: Contains visible page content like text, images, and links.

4. What is the use of the tag?

The html tag wraps the entire HTML document. It tells the browser that the content is in HTML format. It is the root element of all HTML pages.

5. What is the use of the body tag?

The body tag contains everything visible on the web page. This includes text, images, links, and other content. It is displayed in the browser window.
